## Introduction
CoreLocation 简单定位。

1. plist 权限配置 `NSLocationAlwaysAndWhenInUseUsageDescription` `NSLocationWhenInUseUsageDescription`。
2. `CLLocationManager` 进行定位。
3. 经纬度坐标转化真实地理位置。
4. 毛玻璃效果。
