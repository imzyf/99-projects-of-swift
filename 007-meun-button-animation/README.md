## Introduction
菜单键动画。

1. 动画的分割。
2. `CABasicAnimation` `UIBezierPath` `CAAnimationGroup`。
3. `CAAnimationDelegate` 判断动画是否完成。
4. `strokeEnd` `strokeStart` 的运用。
5. `pod 'BezierPathLength', '~> 1.0.0'` 计算 path 长度，获取 path 上的点。
